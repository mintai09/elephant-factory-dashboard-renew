# 회사별 더미 데이터 요약

## 개요

각 회사는 **폐플라스틱(PET, HDPE)**과 **장난감(복합 플라스틱)** 두 가지 폐기물을 수거합니다.

### 폐기물 처리 방식

1. **폐플라스틱 (PET, HDPE)**
   - **100% 섬유 업사이클링**: 조끼, 장갑, 에코백 등으로 제작
   - PET 계수: 2.29 kg CO₂/kg × **2.5 (UF)**
   - HDPE 계수: 3.12 kg CO₂/kg × **2.5 (UF)**

2. **장난감 (혼합 플라스틱)**
   - 재사용 (25~20%): 수리 후 기부 → **3.0 (RBF)** 적용
   - 업사이클링 (55~50%): 키트, 액세서리 → **2.5 (UF)** 적용
   - 재활용 (20~30%): 일반 재활용 → **1.0 (기본)** 적용
   - 혼합 플라스틱 계수: 2.75 kg CO₂/kg

## 1. 코멘토 (IT/기술)

### Q1 2025 수거 내역

#### 폐플라스틱: 800kg
- **PET**: 650kg (사무실, 카페 페트병)
- **HDPE**: 150kg (음료 용기)
- **처리**: 100% 섬유 업사이클링 → 조끼, 장갑 제작

**CO₂ 계산**:
```
(650 × 2.29 × 2.5) + (150 × 3.12 × 2.5) / 1000
= (3,721.25 + 1,170) / 1000
= 4.89 tonnes CO₂
```

#### 장난감: 440kg
- **재사용**: 110kg (25%) - 수리 후 기부
- **업사이클링**: 242kg (55%) - 교육 키트, 액세서리
- **재활용**: 88kg (20%) - 일반 재활용

**CO₂ 계산**:
```
(110 × 2.75 × 3.0) + (242 × 2.75 × 2.5) + (88 × 2.75 × 1.0) / 1000
= (907.5 + 1,663.75 + 242) / 1000
= 2.81 tonnes CO₂
```

### Q1 총 성과
- **총 수거량**: 1,240kg (폐플라스틱 800kg + 장난감 440kg)
- **총 CO₂ 저감**: **7.70 tonnes** (폐플라스틱 4.89 + 장난감 2.81)
- **참여 인원**: 320명
- **순위**: 🥇 1위/4개사

### 특징
- IT 기업으로 사무실 페트병 수거량 많음
- 직원 참여율 가장 높음 (28%)
- 교육 프로그램 활발 (120명)
