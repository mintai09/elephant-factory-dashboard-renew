# 코끼리공장 탄소 저감량 계산 로직

## 1. 기본 개념

폐플라스틱과 장난감(복합 플라스틱)의 탄소 저감량 계산은 **폐기물 유형**과 **순환 경로**에 따라 다르게 적용됩니다.

## 2. 폐기물 유형별 CO₂ 배출계수

폐기물을 소각할 때 발생하는 CO₂ 배출량 (IPCC 2006 GL Volume 5 기준)

| 폐기물 유형 | CO₂ 배출계수 | 설명 |
|------------|-------------|------|
| PET | 2.29 kg CO₂/kg | 페트병 등 단일 재질 플라스틱 |
| HDPE | 3.12 kg CO₂/kg | 고밀도 폴리에틸렌 (용기류) |
| 혼합 플라스틱 | 2.75 kg CO₂/kg | 장난감 등 복합 플라스틱 |

**중요**: 장난감은 여러 종류의 플라스틱이 혼합되어 있어 **2.75 kg CO₂/kg** 계수를 사용합니다.

## 3. 순환 경로별 보정 계수

폐기물을 어떻게 처리하느냐에 따라 탄소 저감 효과가 다릅니다.

### 3.1 RBF (Reuse Benefit Factor)

**계수값**: 3.0

**적용 대상**: 재사용(수리/기부)되는 장난감

**근거**: 재사용은 신제품 생산을 완전히 대체하여 에너지 투입을 대폭 줄이므로, 일반 재활용 대비 3배의 탄소 저감 효과가 있습니다.

**예시**:
- 100kg의 장난감을 수리하여 기부
- 기본 CO₂ 저감량: 100kg × 2.75 = 275 kg CO₂
- 보정 후 저감량: 275 × 3.0 = **825 kg CO₂**

### 3.2 UF (Upcycling Factor)

**계수값**: 2.5

**적용 대상**: 심하게 손상된 장난감을 고품질 재생원료나 업사이클 제품(섬유 원료, 고부가가치 키트 등)으로 전환

**근거**: 단순 재활용보다는 높고(더 많은 버진 소재 대체), 재사용보다는 낮은 중간 수준의 환경 가치를 인정합니다.

**예시**:
- 100kg의 손상된 장난감을 업사이클링
- 기본 CO₂ 저감량: 100kg × 2.75 = 275 kg CO₂
- 보정 후 저감량: 275 × 2.5 = **687.5 kg CO₂**

### 3.3 일반 재활용

**계수값**: 1.0 (기본)

**적용 대상**: 재활용 원료로 전환

**예시**:
- 100kg의 장난감을 재활용
- 기본 CO₂ 저감량: 100kg × 2.75 = 275 kg CO₂
- 보정 후 저감량: 275 × 1.0 = **275 kg CO₂**

## 4. 최종 탄소 저감량 계산 공식

```
총 CO₂ 저감량 = Σ(순환 경로별 폐기물량 × 기본 CO₂ 저감량 × 보정계수)
```

### 4.1 세부 계산식

```
총 CO₂ 저감량 = (재사용량 × CO₂계수 × RBF)
              + (업사이클링량 × CO₂계수 × UF)
              + (일반재활용량 × CO₂계수 × 1.0)
```

### 4.2 장난감 기준 계산식

```
총 CO₂ 저감량 = (재사용량 × 2.75 × 3.0)
              + (업사이클링량 × 2.75 × 2.5)
              + (일반재활용량 × 2.75 × 1.0)
```

## 5. 실제 계산 예시

### 예시 1: 1톤(1,000kg)의 장난감 처리

**비율 가정**:
- 재사용: 20% (200kg)
- 업사이클링: 50% (500kg)
- 일반 재활용: 30% (300kg)

**계산 과정**:
1. 재사용 저감량: 200kg × 2.75 × 3.0 = 1,650 kg CO₂
2. 업사이클링 저감량: 500kg × 2.75 × 2.5 = 3,437.5 kg CO₂
3. 일반재활용 저감량: 300kg × 2.75 × 1.0 = 825 kg CO₂

**총 CO₂ 저감량**: 1,650 + 3,437.5 + 825 = **5,912.5 kg CO₂ = 5.91 tonnes CO₂**

### 예시 2: 코멘토 2025 Q1 실적 (1,240kg 수거)

**비율 가정**:
- 재사용: 25% (310kg)
- 업사이클링: 55% (682kg)
- 일반 재활용: 20% (248kg)

**계산 과정**:
1. 재사용: 310kg × 2.75 × 3.0 = 2,557.5 kg CO₂
2. 업사이클링: 682kg × 2.75 × 2.5 = 4,688.75 kg CO₂
3. 일반재활용: 248kg × 2.75 × 1.0 = 682 kg CO₂

**총 CO₂ 저감량**: 2,557.5 + 4,688.75 + 682 = **7,928.25 kg CO₂ ≈ 7.93 tonnes CO₂**

### 예시 3: PET 병 재활용과 비교

**1톤의 PET 병 수거 (일반 재활용)**:
- 계산: 1,000kg × 2.29 × 1.0 = **2,290 kg CO₂ = 2.29 tonnes CO₂**

**1톤의 장난감 수거 (혼합 처리, 예시 1 기준)**:
- 총 저감량: **5.91 tonnes CO₂**

**차이**: 장난감 순환은 PET 재활용 대비 약 **2.58배** 높은 탄소 저감 효과

## 6. CO₂ 저감량 가중 평균 계산

전체 저감량을 수거량으로 나눈 가중 평균:

```
CO₂ 저감 가중치 = (재사용비율 × 3.0) + (업사이클링비율 × 2.5) + (재활용비율 × 1.0)
```

**예시 1의 경우**:
```
가중치 = (0.20 × 3.0) + (0.50 × 2.5) + (0.30 × 1.0)
       = 0.60 + 1.25 + 0.30
       = 2.15
```

**실제 톤당 저감량**:
```
톤당 CO₂ = 2.75 (기본계수) × 2.15 (가중치) = 5.91 kg CO₂/kg
```

## 7. 데이터 구조 설계 제안

### 7.1 캠페인별 상세 데이터

```javascript
{
  campaignId: 1,
  name: '수달 보호 캠페인',
  period: '2025.03.01 ~ 2025.03.31',
  collection: {
    total: 480,  // kg
    breakdown: {
      reuse: 120,      // 25% - 재사용
      upcycling: 264,  // 55% - 업사이클링
      recycling: 96    // 20% - 일반재활용
    }
  },
  co2Reduction: {
    byType: {
      reuse: 120 * 2.75 * 3.0,      // 990 kg CO₂
      upcycling: 264 * 2.75 * 2.5,  // 1,815 kg CO₂
      recycling: 96 * 2.75 * 1.0    // 264 kg CO₂
    },
    total: 3.069,  // tonnes CO₂
    weightedAverage: 2.15  // 가중 계수
  }
}
```

### 7.2 기업별 Tier 1 KPI 계산

```javascript
{
  carbonReduction: {
    monthly: 2.8,  // tonnes CO₂ (최종 합계)
    breakdown: {
      reuse: 0.99,      // tonnes - 재사용 기여
      upcycling: 1.82,  // tonnes - 업사이클링 기여
      recycling: 0.26   // tonnes - 재활용 기여
    },
    collectionBreakdown: {
      reuse: { weight: 310, ratio: 25 },
      upcycling: { weight: 682, ratio: 55 },
      recycling: { weight: 248, ratio: 20 }
    },
    coefficient: {
      base: 2.75,      // 혼합 플라스틱 기본계수
      weighted: 2.15   // 가중 평균 계수
    }
  }
}
```

## 9. 구현 시 주의사항

1. **캠페인별 순환 경로 비율 추적 필요**
   - 각 캠페인마다 재사용/업사이클링/재활용 비율이 다를 수 있음
   - 실제 데이터 수집 시 이 비율을 정확히 기록해야 함

2. **기본 계수는 폐기물 유형별로 다름**
   - 장난감: 2.75 kg CO₂/kg
   - PET 병: 2.29 kg CO₂/kg
   - HDPE: 3.12 kg CO₂/kg

3. **보정 계수 적용 우선순위**
   - 재사용 (3.0) > 업사이클링 (2.5) > 일반재활용 (1.0)
   - 환경 임팩트가 큰 순서대로 우선 배정

4. **월간/분기별 집계**
   - 각 캠페인의 CO₂ 저감량을 합산
   - 가중 평균 계수를 사용하여 종합 성과 계산

## 10. 목표 설정 기준

**KPI #1: 탄소 저감 기여도 목표**

| 등급 | 월간 목표 | 연간 목표 |
|------|----------|----------|
| 기본 | 1.5 tonnes 이상 | 18 tonnes |
| 우수 | 3.0 tonnes 이상 | 36 tonnes |
| 탁월 | 5.0 tonnes 이상 | 60 tonnes |

**달성률 계산**:
```
달성률 = (실제 저감량 / 목표 저감량) × 100
```

**예시**: 월 2.8 tonnes 달성 시
- 기본 대비: 2.8 / 1.5 × 100 = 187% (초과 달성)
- 우수 대비: 2.8 / 3.0 × 100 = 93% (거의 달성)
- 탁월 대비: 2.8 / 5.0 × 100 = 56%

## 11. 데이터 검증

실제 구현 시 다음 항목들을 검증해야 합니다:

1. **합계 검증**
   ```
   총 수거량 = 재사용량 + 업사이클링량 + 재활용량
   ```

2. **비율 검증**
   ```
   재사용비율 + 업사이클링비율 + 재활용비율 = 100%
   ```

3. **저감량 검증**
   ```
   총 CO₂ = 각 경로별 저감량의 합
   평균 kg당 저감량 = 총 CO₂ / 총 수거량 (kg 기준)
   ```
