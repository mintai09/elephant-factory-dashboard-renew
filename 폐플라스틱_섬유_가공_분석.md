# 폐플라스틱 섬유 가공 (조끼, 장갑) - 업사이클링 vs 재활용 분석

## 질문
폐플라스틱(PET, HDPE 등)을 섬유로 가공하여 "조끼", "장갑" 등으로 만드는 것은 **업사이클링**인가, **재활용**인가?

## 결론
**업사이클링 (Upcycling)** 으로 분류됩니다.

## 근거

### 1. 업사이클링의 정의

**업사이클링 (Upcycling)**:
- 폐기물을 **원래보다 더 높은 가치의 제품**으로 재탄생시키는 과정
- 단순 재활용보다 **품질과 가치가 향상**됨
- 환경 임팩트: 더 많은 버진 소재(virgin material) 대체

**일반 재활용 (Recycling)**:
- 폐기물을 **동일하거나 낮은 등급의 원료**로 전환
- 예: PET 병 → 저등급 PET 펠릿

### 2. 폐플라스틱 → 섬유 가공이 업사이클링인 이유

#### 2.1 가치 향상
- **원재료**: 폐플라스틱 병, 용기 (버려질 폐기물)
- **최종 제품**: 조끼, 장갑 (기능성 의류)
- 가치 상승: 폐기물(~0원) → 의류 제품(수만원)

#### 2.2 기능 향상
- 단순한 플라스틱 용기 → 보온/방수 기능을 가진 섬유 제품
- 새로운 용도로 재탄생 (container → clothing)

#### 2.3 버진 소재 대체
- 폴리에스터 섬유는 원래 석유에서 추출한 버진 PET로 제조
- 폐플라스틱 섬유는 이를 **100% 대체**
- 높은 환경 가치

### 3. 섬유 가공 프로세스

```
폐플라스틱 (PET 병, HDPE 용기)
    ↓ 수거 및 분류
    ↓ 세척 및 파쇄
    ↓ 용해 및 방사 (spinning)
    ↓ 섬유 (fiber) 생산
    ↓ 직조 (weaving)
    ↓ 재단 및 봉제
    ↓
조끼, 장갑, 가방 등 업사이클 제품
```

### 4. 탄소 저감 효과 비교

#### 4.1 일반 재활용 (기본 계수 1.0)
```
폐플라스틱 1kg → 재생 펠릿 1kg
CO₂ 저감 = 1kg × 2.75 (HDPE 기준 3.12) × 1.0
         = 2.75 kg CO₂
```

#### 4.2 섬유 업사이클링 (UF 계수 2.5)
```
폐플라스틱 1kg → 섬유 → 조끼/장갑
CO₂ 저감 = 1kg × 2.75 × 2.5
         = 6.875 kg CO₂
```

**차이**: 업사이클링은 일반 재활용 대비 **2.5배** 높은 탄소 저감 효과

### 5. 코끼리공장 KPI 문서 기준

문서 내용 (p.2):
> **UF (Upcycling Factor) = 2.5**
>
> "심하게 손상된 장난감을 고품질 재생원료나 업사이클 제품(예: **섬유 원료**, 고부가가치 키트)으로 전환할 때 부여하는 가중치"

**명시적으로 "섬유 원료"를 업사이클링으로 분류**하고 있습니다.

## 6. 폐기물 유형별 섬유 가공 가능성

| 폐기물 유형 | 섬유 가공 가능 | 주요 용도 | CO₂ 계수 | UF 적용 |
|------------|--------------|----------|---------|---------|
| PET 병 | ✅ 매우 적합 | 폴리에스터 섬유 (조끼, 장갑, 가방) | 2.29 | 2.5 |
| HDPE 용기 | ⚠️ 제한적 | 특수 섬유, 부직포 | 3.12 | 2.5 |
| 혼합 플라스틱 (장난감) | ❌ 어려움 | 복합 플라스틱으로 섬유화 곤란 | 2.75 | - |

**중요**:
- **PET 폐플라스틱 → 섬유**: 업사이클링 (UF 2.5)
- **장난감 (혼합 플라스틱)**: 섬유 가공 어려움, 다른 업사이클링 방식 필요

## 7. 데이터 구조 설계

### 7.1 폐기물 유형 분리

코끼리공장은 두 가지 주요 폐기물을 수거:

1. **폐플라스틱 (PET, HDPE)**
   - 순환 경로: 섬유 업사이클링 → 조끼, 장갑 등
   - CO₂ 계수: PET 2.29, HDPE 3.12
   - 보정 계수: UF 2.5

2. **장난감 (혼합 플라스틱)**
   - 순환 경로: 재사용, 업사이클링(다른 방식), 재활용
   - CO₂ 계수: 2.75
   - 보정 계수: RBF 3.0 (재사용), UF 2.5 (업사이클링), 1.0 (재활용)

### 7.2 캠페인별 데이터 구조 제안

```javascript
{
  campaignId: 1,
  name: '수달 보호 캠페인',
  period: '2025.03.01 ~ 2025.03.31',

  // 폐기물 수거 내역
  wasteCollection: {
    // 1. 폐플라스틱 (병, 용기)
    wastePlastic: {
      total: 280,  // kg
      byType: {
        pet: 180,   // kg - PET 병
        hdpe: 100   // kg - HDPE 용기
      },
      // 모두 섬유로 업사이클링
      processing: {
        fiberUpcycling: 280  // 100% - 조끼, 장갑 등
      },
      co2Reduction: {
        pet: 180 * 2.29 * 2.5,    // 1,030.5 kg CO₂
        hdpe: 100 * 3.12 * 2.5,   // 780 kg CO₂
        total: 1.81              // tonnes CO₂
      }
    },

    // 2. 장난감 (복합 플라스틱)
    toys: {
      total: 200,  // kg
      // 순환 경로별 분배
      processing: {
        reuse: 50,        // 25% - 수리 후 기부
        upcycling: 110,   // 55% - 키트, 액세서리 등
        recycling: 40     // 20% - 일반 재활용
      },
      co2Reduction: {
        reuse: 50 * 2.75 * 3.0,       // 412.5 kg CO₂
        upcycling: 110 * 2.75 * 2.5,  // 756.25 kg CO₂
        recycling: 40 * 2.75 * 1.0,   // 110 kg CO₂
        total: 1.28                   // tonnes CO₂
      }
    }
  },

  // 총 성과
  totalCollection: 480,  // kg (280 + 200)
  totalCO2Reduction: 3.09,  // tonnes (1.81 + 1.28)
  participants: 120
}
```

## 8. 회사별 더미 데이터 설계 가이드

### 8.1 코멘토 (IT/기술, 참여 인원 많음)

**특징**: 직원이 많고 참여율 높음, 페트병 수거 많음

```javascript
{
  // Q1 2025 총 수거량: 1,240kg
  wasteCollection: {
    wastePlastic: {
      total: 800,  // 64.5%
      pet: 650,    // 페트병 (카페, 사무실)
      hdpe: 150,   // HDPE 용기
      fiberUpcycling: 800  // 100% 섬유 업사이클링
    },
    toys: {
      total: 440,  // 35.5%
      reuse: 110,       // 25% - 수리 후 기부
      upcycling: 242,   // 55% - 업사이클 제품
      recycling: 88     // 20% - 재활용
    }
  }
}
```

**CO₂ 계산**:
- 폐플라스틱: (650×2.29 + 150×3.12) × 2.5 = 4,883 kg CO₂
- 장난감: (110×2.75×3.0) + (242×2.75×2.5) + (88×2.75×1.0) = 2,405 kg CO₂
- **총 7.29 tonnes CO₂**

### 8.2 SK이노베이션 (에너지/화학)

```javascript
{
```

**CO₂ 계산**: 약 **5.8 tonnes CO₂**

## 10. 결론 및 권장사항

### 10.1 분류 기준

✅ **업사이클링 (UF 2.5 적용)**:
- 폐플라스틱 → 섬유 → 조끼, 장갑
- 손상 장난감 → 고부가가치 키트, 액세서리
- 폐플라스틱 → 펠릿 → 디자인 소품

✅ **재사용 (RBF 3.0 적용)**:
- 장난감 수리 → 기부/재판매
- 상태 좋은 제품 그대로 전달

✅ **일반 재활용 (1.0 적용)**:
- 저등급 플라스틱 펠릿
- 단순 원료 재가공

### 10.2 데이터 수집 항목

각 기업은 다음을 보고해야 함:
1. **폐플라스틱** 수거량 (PET, HDPE 구분)
2. **장난감** 수거량
3. 각각의 **순환 경로** 비율
4. **최종 제품** 생산 실적
